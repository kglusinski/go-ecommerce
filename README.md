# Go E-Commerce Application
This is example e-commerce application written in Go. 
It follows the [Clean Architecture](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html) 
and [Hexagonal Architecture](https://alistair.cockburn.us/hexagonal-architecture/) principles.

## Features
### Non-functional
- [x] REST API
- [ ] gRPC API
- [ ] JWT authentication
- [ ] Docker
- [ ] Event sourcing
### Functional
- [x] add product
- [ ] list products
- [x] get product
- [ ] update product
- [ ] delete product
- [x] add product to cart
- [x] list products in cart
- [x] remove product from cart
- [ ] checkout cart
- [ ] list orders
